# Store inventory

Store inventory is a Flask app for store products using a mongodb as database.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the followings modules.

```bash
pip install flask
```
```bash
pip install pymongo
```
```bash
pip install flask_pymongo
```
